About Cycle Tracker

What it does

Cycle Tracker logs menstrual periods and visualizes the four phases of the cycle across several months. It's designed for the person who has the cycle or their partner — whoever is doing the tracking. As more cycles are logged, the app automatically calculates the actual average cycle length and adjusts predictions to match.

ColorPhaseTypical length
CoralMenstrual~5 days
TealFollicular~9 days
PurpleOvulation~3 days
AmberLuteal~11 days

How to use it

Add to your iPhone home screen

  1. Open the app in Safari
  2. Tap the Share button (box with arrow at the bottom)
  3. Tap Add to Home Screen
  4. Tap Add

Logging periods

Tap any day on the calendar to open a menu. From there you can:

  • Log that day as a period start
  • Set it as the end of an open period
  • Update the end date of a nearby logged period
  • Remove a period entry

You can also log periods manually in the Log tab.

Cycle length calculation

After 3 or more periods are logged, the app calculates your actual average cycle length automatically. The cycle days stat card will show a blue calculated badge when this is active.

Syncing between devices

By default, data is stored only in your browser on that device. To sync across devices — useful if both partners want to view the calendar — go to Settings → Sign in with Google. Data is saved to a file called cycle-tracker-data.json in a shared Google Drive account.

Backing up your data

Go to Settings → Export JSON to download a backup. You can re-import it on any device using Settings → Import JSON.

Privacy

🔒 Your data never leaves your device unless you choose to enable Google Drive sync.

  • No server. The app is a single HTML file — there is no backend.
  • No analytics. No tracking, no cookies, no telemetry of any kind.
  • No account required. You never sign up for anything.
  • Data stays on your device. Period data is stored in your browser's local storage and never transmitted anywhere. Neither partner's data is accessible to anyone else.
  • Google Drive sync is optional. If you use it, data goes to a Google Drive account you control — shared between partners if you choose. The app developer cannot see it.
  • Open source. Every line of code is visible at github.com/rharder/cycle-tracker.
